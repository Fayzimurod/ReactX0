# Tic-Tac-Toe Game ❌⭕

A classic Tic-Tac-Toe game built with React. Play against a friend in this interactive and visually appealing game with real-time win detection and reset functionality.

## ✨ Features

- 🎮 Two-player turn-based gameplay
- ⭕ Circle (O) and ❌ Cross (X) icons
- 🏆 Automatic win detection for all combinations
- 🔄 Real-time winner announcement
- 🎯 Game lock after win (prevents further moves)
- 🔁 Reset button to start new game
- 📱 Responsive design
- 🎨 Dynamic title updates showing winner

## 🛠️ Technologies Used

- React.js
- CSS3
- JavaScript (ES6+)
- React Hooks (useState, useRef)

## 🎯 How to Play

1. **Start the game:** Players alternate turns automatically
2. **First player:** Circle (O) starts the game
3. **Second player:** Cross (X) plays next
4. **Win conditions:** Get three in a row:
   - Horizontally
   - Vertically
   - Diagonally
5. **Game end:** Winner announced or board locks
6. **New game:** Click "Refresh" button to reset

## 🎨 Styling Features
